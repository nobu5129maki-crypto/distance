スマート・レンジファインダー (Smart Rangefinder - Manual Edition)

スマートフォンのカメラと簡単な三角法を利用して、対象物までの距離を即座に測定できるWebアプリケーションです。

🌟 特徴

センサー不要: ジャイロセンサーなどの高度なAPIを使用せず、画面のタッチ操作のみで直感的に計測が可能です。

プライバシー配慮: カメラ映像はブラウザ内での計算にのみ使用され、外部サーバーに送信されることはありません。

リアルタイム表示: 画面をスライドさせるだけで、瞬時に距離が再計算されます。

モバイル最適化: PWAのようなフルスクリーン体験を提供し、操作を妨げない上部UIを採用しています。

📐 計測の仕組み

このアプリは「三角法」の原理を利用しています。

カメラの高さ ($h$): 地面から目線（スマホのレンズ）までの高さ。

俯角 ($\theta$): 水平線から対象物の接地点（足元）を見下ろした時の角度。

以下の公式を用いて、対象物までの水平距離 ($d$) を算出します。

$$d = \frac{h}{\tan(\theta)}$$

🚀 使い方

URLを開く: カメラの使用許可を求められるので「許可」を選択してください。

高さを設定: 画面上部の入力欄に、自分の目線の高さ（例：1.5m）を入力します。

水平を保つ: 画面中央の 白い点線 を、遠くの水平線や自分と同じ高さにある目印に合わせます。

ターゲットを合わせる: 画面を上下にスライドさせ、 赤い実線 を測りたい物体の「地面に接している部分」に合わせます。

結果を確認: 上部に表示されている数値が、あなたからその物体までの距離（メートル）です。

🔧 カスタマイズ

お使いのデバイスによってカメラの画角（FOV）が異なる場合があります。計測結果が常にズレる場合は、rangefinder.html 内の以下の定数を調整してください。

// デバイスの垂直FOV (標準的なスマホは約60度前後)
const VERTICAL_FOV = 60;
