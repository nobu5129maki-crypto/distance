フィールド・メジャー Pro (Field Distance Pro)

スマホの傾斜センサー（ジャイロセンサー）を利用して、対象物までの距離をリアルタイムで測定する野外専用のWebアプリケーションです。

🎯 特徴

野外特化型UI: 直射日光下でも視認性が高いハイコントラスト（ブラック×蛍光グリーン）なデザイン。

高精度計測: 三角関数を用いた計算ロジックにより、AR機能が不安定な場所でも安定した計測が可能。

デジタル水平器: デバイスの左右の傾きを視覚化し、測定誤差を最小限に抑えます。

ハードウェアズーム: 対応端末ではスライダーによる光学/デジタルズーム操作が可能。

測定ログ機能: 記録ボタンで測定結果をタイムスタンプと共にリスト保存。

PWA対応考慮: シンプルな1ファイル構成で、オフライン環境での動作も視野に入れた設計。

🚀 使い方

GitHub Pages等にアップロード: 本アプリはカメラとセンサーを使用するため、必ず HTTPS環境 で実行する必要があります。

システム開始: 画面下のボタンを押し、カメラと動作センサーのアクセスを許可します。

カメラ高の設定: 自分がスマホを構えた時の地面からの高さ（例: 1.5m）を入力します。

ターゲットを狙う: 画面中央の緑の点を、測りたい対象の「足元（接地面）」に合わせます。

水平を確認: 中央の横線が緑色になるよう、スマホを左右に傾けず垂直に保持します。

記録: 右下のボタンで測定値を保存します。

🛠 技術仕様

言語: HTML5, JavaScript (Vanilla JS)

スタイリング: Tailwind CSS

API:

getUserMedia (カメラアクセス)

DeviceOrientationEvent (傾斜センサー)

applyConstraints (ズーム制御)

⚠️ 注意事項

環境: ブラウザは iOS Safari または Android Chrome を推奨します。

制限: iPhone (iOS) ではブラウザの制限により、JavaScriptからのズーム制御が効かない場合があります。その際は指でのピンチ操作でズームしてください。

精度: 本アプリは推定値を算出するものであり、厳密な測量を保証するものではありません。
